## 什么是工程化

- 工程化即系统化、模块化、规范化的一个过程。

- 工程化要解决的是如何提高整个系统生产效率。

- 与其说软件工程是一门科学，不如说它更偏向于管理学和方法论

前端工程化主要应该从规范化、组件化、模块化、自动化、系统化 五个方面来思考：


### 从规范化角度上来说：


#### (1) 制定编码规范

#### (2) 制定文件命名规范

#### (3) 制定组件的分割规范

#### (4) 工作流规范


### 从模块化角度上来说：

- 前端模块化方案

- 前端模块化管理


### 从组件化角度上来说：

- 组件管理

- 组件划分

- 组件性能
